happen on `io.SshChannel` object. The set of *active* events is accessible
via `io.SshChannel.m_activeEvents`. :ref:`More...<details-typeio_sshevents>`

Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This enumeration contains codes of all possible events that might
happen on `io.SshChannel` object. The set of *active* events is accessible
via `io.SshChannel.m_activeEvents`.
