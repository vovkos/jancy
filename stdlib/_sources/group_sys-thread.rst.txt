Global Functions
----------------

.. index:: pair: function; getCurrentThreadId
.. _doxid-functionsys_getcurrentthreadid:

.. ref-code-block:: jancy
	:class: doxyrest-title-code-block

	:ref:`uintptr_t<doxid-typedefuintptr_t>` sys.getCurrentThreadId()

Returns the thread identifier (TID) of the current thread.

Translates to ``GetCurrentThreadId`` on Windows and ``pthread_self`` on
POSIX.

.. index:: pair: function; getCurrentProcessId
.. _doxid-functionsys_getcurrentprocessid:

.. ref-code-block:: jancy
	:class: doxyrest-title-code-block

	:ref:`uint_t<doxid-typedefuint_t>` sys.getCurrentProcessId()

Returns the process identifier (PID) of the current process.

Translates to ``GetCurrentProcessId`` on Windows and ``getpid`` on
POSIX.
