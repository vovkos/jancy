Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This union is a container for both ``IP4`` and ``IP6`` addresses.

All methods and/or properties of `io.Socket` class which accept or return
addresses, use ``io.SocketAddress`` arguments.
