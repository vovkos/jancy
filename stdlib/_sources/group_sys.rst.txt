Properties

.. index:: pair: property; g_env
.. _doxid-property_sys_g_env:

.. ref-code-block:: jancy
	:class: doxyrest-title-code-block

	:ref:`string_t<doxid-typedef_string_t>` indexed property sys.g_env(:ref:`string_t<doxid-typedef_string_t>` name)

Provides access to the system environment variable table.

Reading from this property results in a call to ``getenv`` standard C
library function.

Writing to this property translates to ``setenv`` on Linux/Mac and
``_putenv`` on Windows.
