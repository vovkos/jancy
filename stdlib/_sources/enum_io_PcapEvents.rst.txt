bitflag enum io.PcapEvents
==========================

Overview
~~~~~~~~

This enumeration contains codes of all possible events that might
happen on `io.Pcap` object. The set of *active* events is accessible
via `io.Pcap.m_activeEvents`. :ref:`More...<details-typeio_pcapevents>`
