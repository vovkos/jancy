Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This struct holds information about the serial port device.

A list of serial port descriptions is returned by
`io.enumerateSerialPorts`.



.. rubric:: See also:

`io.enumerateSerialPorts`
