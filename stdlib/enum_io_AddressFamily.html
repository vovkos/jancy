enum io.AddressFamily

Overview

This enumeration contains possible values for socket address types (families). More…

enum AddressFamily: uint16_t {
    Undefined = 0,
    Ip4       = 2,
    Ip6,
};

Detailed Documentation

This enumeration contains possible values for socket address types (families).

io.AddressFamily values map to AF_XXX platform constants. At the moment, only IP4 and IP6 address families are supported.

See also:

io.SocketAddress

Enum Values

Undefined

Used to designate an uninitialized value.

Ip4

Specifies IP4-address; maps to AF_INET platform constant.

Ip6

Specifies IP6-address; maps to AF_INET6 platform constant.
